🚀 Autonomous AI Research Agent

An Autonomous AI Research Agent that can search the internet, generate a research report, critique its own work, and iteratively improve the report until it reaches high quality.

This project demonstrates how modern AI agents can reason, evaluate, and self-improve without human intervention.

🧠 How It Works

The system follows a multi-agent workflow:

Research Agent

Searches the internet for relevant information.

Writer Agent

Generates a structured research report from the collected sources.

Critic Agent

Evaluates the quality of the report and assigns a score from 1–10.

Improver Agent

Uses feedback from the critic to improve the report.

Self-Improvement Loop

The system continues improving the report until the quality score ≥ 9.
